## 🌟 Project Overview
A Windows Forms application designed for Finnish users to calculate trip fuel costs with precision. 

**Key Highlights:**
- 🇫🇮 Finnish UI localization (suomi)
- ⚡ Dual measurement modes: L/100km (European) and L/km
- 💰 Euro (€) currency formatting
